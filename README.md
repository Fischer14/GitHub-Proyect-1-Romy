## Acerca del Proyecto

<img src="./resources/images/readMe/screenshot.png" alt="Logo" width="100%" height="50%">

El propósito de este proyecto es dearrollar una página web para la empresa Circle, que cree una solución digital que realmente impulse el crecimiento y presencia en línea de su negocio. Una página web completamente personalizada y adaptada a las necesidades de su empresa, que no solo cautivará a sus clientes, sino que también brindará una experiencia de usuario excepcional.
